◆ cose_sign_encode()

ssize_t cose_sign_encode ( cose_sign_t *  sign,
uint8_t *  buf,
size_t  len,
uint8_t **  out,
cn_cbor_context *  ct 
)

cose_sign_sign signs the data from the sign object with the attached signers. The output is placed in the supplied buffer, starting at the position indicated by the out parameter.

This function uses the buffer as scratch space to first calculate all the signatures. Therefor this buffer should be large enough to contain the headers, the payload, the additionally authenticated data and the signatures at the same time. This is a limitation caused by how the COSE signatures to be generated and how crypto libraries require their message as one continuous block of data.

Parameters
signSign struct to encode
bufBuffer to write in
[out]outPointer to where the COSE sign struct starts
lenSize of the buffer to write in
ctCN_CBOR context for cbor block allocation
Returns
The number of bytes written
Negative on error
